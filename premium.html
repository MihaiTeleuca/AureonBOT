Planned usage

What Gems are meant to unlock later.

Aureon Premium is planned around optional upgrades. The goal is to make support feel useful and fair, while keeping the bot’s main systems available for communities that use the free version.

👑

Server VIP

Gems can later be redeemed for VIP access on servers that enable Aureon’s premium system. Server owners may decide what VIP includes for their community.

🪪

Profile extras

Future profile perks may include visual badges, profile themes, small identity upgrades and other cosmetic details that help members personalize their presence.

🎁

Community rewards

Gems can support special server events, limited rewards, giveaway entries or controlled economy bonuses depending on how each server configures Aureon.

Flow

A simple process inside Discord.

The premium system is planned to be easy to follow. Users should be able to understand what they own, what they can redeem and what is active on their server.

Principles

Premium should feel useful, not forced.

Aureon’s premium direction is built around transparency. The free version should remain helpful, while Gems provide optional support and extra comfort for users who want more.

🧭

Clear value

Each Gems package should be understandable before purchase. Members should know what they are buying and how it may be used later.

⚖️

Fair server balance

Premium rewards should avoid damaging the server economy. Server owners can shape VIP benefits in a way that fits their community.

For server owners

VIP can become part of your server’s identity.

When VIP support is enabled, server owners can connect premium rewards with roles, events, economy bonuses or visual identity. The goal is to make premium feel like a natural server feature, not a separate confusing system.
